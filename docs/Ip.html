IP Addresses,Packets,and Routing

Internet Protocol

The Internet Protocol is a communications protocol for relaying datagrams across network boundaries. It sets up internetworking between many different devices and establishes the Internet itself.

IP Addresses

An IP addresses is a numerical label given to each device connected to a computer network that uses Internet Protocol. An example of an IP address is:168.212. 226.204

Packets

Network packets are used in computer networking. Its a unit of data that contains information that is made up of control information and user data. The sender makes sure the packet goes to correct IP address to deliver the information efficiently.

Routing

In a network, routing is in charge of selecting a path for traffic in a network. Usually would send the packet data around the networks.
