HTTP and HTTPS Protocol

HTTP

An application layer protocol for distributed, collaborative, and hypermedia information Systems. It is the foundation of data communication for the World Wide Web. The development was initiated by Tim Berners-Lee the man who invented the World Wide Web. It functions as a request- response protocol in a client server computer model.

HTTPS

HTTPS is an extention to the HTTP protocol. It is popular on the internet because it provides secure communication over a computer network. HTTPS gives protection of the privacy of exchanged data while it is being transferred. It is mostly used on websites where payment transactions take place. Websites such as, Amazon and Ebay use HTTPS protocol.
